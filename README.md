# 互动作品集

这是一个基于 Next.js 的个人作品集项目，主要收录我做过的互动 H5 页面，包括测试、占卜、轻游戏和趣味工具。

如果这份仓库是从简历链接点进来的，建议优先查看以下几个页面：

- `天选之城`：城市倾向测试
- `职场前世测试`：古代身份/性格气质测试
- `剧系恋人`：角色匹配测试
- `逆流鹅上`：轻互动游戏
- `带薪摸鱼计算器`：趣味工具页

## 项目内容

当前已经接入的页面包括：

- 首页作品集导航
- 职场前世测试
- 剧系恋人
- 每日江湖签
- 命运图谱
- 逆流鹅上
- 天选之城
- 带薪摸鱼计算器
- 刀盾狗大作战（静态页面资源）

## 技术栈

- Next.js
- React
- TypeScript
- CSS Modules

## 本地运行

```bash
npm run dev
```

打开 [http://localhost:3000](http://localhost:3000) 查看开发效果。

## 生产构建

```bash
npm run build
npm run start
```

## 目录结构

- `pages/`：页面路由
- `styles/`：页面样式
- `config/`：测试题目与配置数据
- `assets/`：项目图片资源
- `public/`：静态资源与素材
- `lib/`：工具函数

## 说明

这个仓库当前的重点不是做通用组件库，而是集中展示我完成过的互动页面成品，因此页面之间的视觉风格和交互类型会有明显差异。
